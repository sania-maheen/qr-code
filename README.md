# QR Code Generator

A simple QR Code Generator built using **HTML, CSS, and JavaScript**.
This project allows users to enter text or a website link and instantly generate a QR code that can be scanned using a mobile device.

## Features

* Generate QR codes from text or URLs
* Automatically formats website links to ensure they open correctly
* Clear button to reset the input and QR code
* Simple and responsive user interface

## Technologies Used

* HTML
* CSS
* JavaScript
* QRCode.js library

## How to Use

1. Enter text or a website URL in the input field.
2. Click the **Generate** button.
3. A QR code will be generated instantly.
4. Scan the QR code using your phone camera to access the encoded link or text.

## Example

Typing `google` will automatically convert to `https://google.com` and generate a QR code that opens **Google** in a browser.
